How To Add New Art

Step-by-step guide to uploading drawings to your gallery using Google Drive.

Step-by-Step Guide
  1. Take a clear photo of your drawing. Good lighting makes a huge difference — natural light near a window works great. Make sure the whole drawing is in frame.
  2. Upload to Google Drive. Go to drive.google.com, click + New → File upload, and select your photo.
  3. Get the shareable link. Right-click the uploaded image → Share → Change to "Anyone with the link" → Copy link.
  4. Extract the File ID. The link looks like:
    https://drive.google.com/file/d/1aBcDeFgHiJkLmNoPqRsTuV/view?usp=sharing
    The bold part between /d/ and /view is your File ID.
  5. Test your File ID using the tester below ↓ — paste it in and check the image loads correctly.
  6. Add to your gallery — go to the page, choose the right room, click "Add Artwork", and paste the File ID.
  7. Click "Generate gallery-config.js" and download/replace the file in your gallery folder. Your gallery updates instantly!

Go Live

How to put your gallery on the internet for free so anyone can visit it.

Option A — GitHub Pages Free · Recommended
  1. Create a free account at github.com
  2. Click + New repository → name it kwesipee-gallery → set to Public → Create
  3. Upload your 3 files: index.html, gallery-config.js, admin.html
  4. Go to Settings → Pages → Source → Deploy from branch → main
  5. Your gallery is live at: https://yourusername.github.io/kwesipee-gallery

💡 To update your gallery later: just replace the gallery-config.js file on GitHub and the live site updates in ~1 minute.

Option B — Netlify Free · Even Easier
  1. Go to netlify.com → Sign up free
  2. Drag and drop your gallery folder onto the Netlify dashboard
  3. Get a live URL instantly — e.g. kwesi-gallery.netlify.app
  4. To update: drag the folder again with the new gallery-config.js
